Auto Light Control System

  • This control system is used to automatically controls lights according to user behavior and it includes several light units, an MQTT server, and an AWS server.

  • The light units are controlled remotely, via cellphone, and each light unit is capable of obtaining the user’s position by RSSI.

  • All light units are connected to a local server, using MQTT message broker, and the local server controls the light, based on AWS data and feedback from the light units.

  • A camera is connected to the controlling server, which processes the image input using TensorFlow to recognize user’s behavior.

  • The control server learns the light pattern associated with the user’s behavior and adjusts the light based on the data.

Warehouse Application

  • This C# application can help users find the shortest path to pick up items in a warehouse.

  • The program loads map data in any excel format and displays a simply map in GUI.

  • Users can either type in the item they want or load the item list in text format. The program searches the path and displays the order for items to be pick by following the path.

  • There are two algorithms to find the path: one is nearest neighbor, and the other is Branch and Bound. When the program starts looking for the path, it concurrently runs both algorithms and returns the result when one of the algorithms is complete.
